# aegis-comply
Демонстрационный прототип цифровой платформы для первичной оценки санкционных, экспортно-контрольных, юрисдикционных и финансовых рисков внешнеэкономической сделки. MVP показывает не чат-бот, а самостоятельную web-платформу с логикой скоринга, комплаенс-досье и моделью дальнейшей интеграции с правовыми источниками.
